Welcome to ShadowNet Monolith

This is a vulnerable machine simulation containing multiple services. This specific web application is the Shadow-Flask gateway.

Available Vulnerabilities:

  • SQL Injection: Try bypassing the login form.
  • Stored XSS: Can you inject a script into the guestbook?
  • IDOR: Access other profiles once logged in.
  • RCE: Execute commands through the network utility.
  • SSRF: Access internal services (Redis, LDAP, MySQL) from the fetch tool.
  • SSTI: Manipulate the greeting template.

Other Exposed Services:

  • Port 80: Apache Gateway (Shellshock RCE)
  • Port 21: FTP (Anonymous Access)
  • Port 2222: SSH (service:password)
  • Port 3306: MySQL (root:root)
  • Port 6379: Redis (No Auth)
  • Port 389: LDAP (Anonymous Bind)
